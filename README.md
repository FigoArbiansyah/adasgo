### How to use

To run this project

```bash
git clone https://github.com/FigoArbiansyah/adasgo.git
```

```bash
cd adasgo
```

```bash
npm install
```

```bash
npx tailwindcss -i ./src/input.css -o ./dist/css/output.css --watch
```
